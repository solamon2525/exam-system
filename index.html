📖 วิธีสร้าง Firebase Project (ทำครั้งเดียว)
  1. ไปที่ console.firebase.google.com
  2. กด Add project → ตั้งชื่อ → กด Continue → Create project
  3. เมนูซ้าย → Firestore Database → Create database → Start in test mode → Done
  4. ไอคอนเฟือง ⚙️ → Project Settings → แท็บ General → เลื่อนลง → Add app → เลือก Web </>
  5. ตั้งชื่อ app → Register → คัดลอก firebaseConfig → วางในช่องด้านล่าง
